# iumi Landing Page

Static landing page for **イウミ｜ウェブデザイン**.

## Pages

- `index.html` — Main landing page
- `portfolio.html` — Portfolio
- `banners.html` — Banner work samples
- `memo.html` — Site production notes

## Local preview

Open `index.html` in a browser, or serve the folder with any static file server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Contact form

The contact form uses [FormSubmit](https://formsubmit.co/) and sends to `ciao.iumiweb@gmail.com`.
